## 입문에서 쓰는 Git 명령어
프로젝트 시작
- git init - 맨 땅에서 시작할 때
- git clone - 다른 프로젝트를 복사해서 시작할 때

커밋하기 (코드 저장하기)
- git add . - 전체 파일을 커밋
- git commit -m "커밋 메시지" - 커밋 메시지와 함께 커밋함

푸시하기 (깃허브에 코드를 업로드)
- git push
